QUESTION 1

What distinguishes normal HTML elements from semantic HTML elements?

Answer

  • Normal HTML elements don't have any inherent meaning or structure, while semantic HTML elements convey the meaning of the content they contain.
  • Normal HTML elements are used to create the basic structure of a web page, while semantic HTML elements provide a more structured way of organizing content.
  • Semantic HTML elements provide extra information that can be used by assistive technologies, like screen readers, to better understand the content on the page.
Examples of normal HTML elements are:
  • div
  • span
  • img
Examples of semantic HTML elements are:
  • article
  • nav
  • section

QUESTION 2

Explain the disparity between closing tags and self-closing tags in HTML.

  • Closing tags are used to close an HTML element and mark the end of it's contents. Example, the paragraph tag "p".
  • Self-closing tags are used for HTML elements that don't have any contents, like the image tag "img"

QUESTION 3

What distinguishes inline elements from block elements in HTML?

  • Inline elements are displayed in the same line as other elements. Examples include the a tag, strong, and em. These elements are typically used to style or format text, such as making it bold or italized
  • Block elements are displayed on their own line and typically take up the full width of their container. Examples include div,p,and h1. These elements are typically used to structure the content of the web page into differnt sections.
